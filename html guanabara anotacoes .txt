computador se comunica por sinais
0 e 1 - digitos binarios
1 bit é um 0 ou um 1
8bits é o conjunto minimo pra q se represente um dado
o nome dessa sequencia é byte
1 byte é uma sequencia de 8bits
1 byteé a porção minima para representar uma informacao
exemplo: 01000001 = A
a letra A, seguindo a tabelado codigo multibyte UTF-8
é representada por essa sequencia de bits
Esse codigo multibyte utiliza de 1 a 4 bytes para
representar letras, simbolos, emojis
os 0 e 1 sao dois estados, chamado de "onda quadrada"
onda binarios = onda quadrada
o computador só entende essas ondas binarias
na computação nos temos os multiplos do byte
8bits = 1byte, 1024bytes = 1KB
1024KB = 1MB, 1024MB = 1GB
porque na computação é 1024?
porque ele tabalha com a base 2, e nao 10 como a nossa

quero acessar instagram
instagram é um conjunto de arquivos
onde estão esses arquivos?
estão em um servidor, ou melhor, um conjunto deles
esses servidores são identificados com um numero
quando vc se conecta tambem recebe um numero de identificacao
esses são os numeros IPs
nesse caso, um numero IP publico
existe tmb o IP local, o do seu aparelho
toda vez que vc acessaria o instagram,
teria que digitar o numero IP publico do servidor,
o que seria um problema, ja que eles sempre mudam.
pra isso a interner criou uma "agenda eletronica"
que é o servidor DNS - Sistema de Nomes de Domínio
Exemplo de dominio - instagra.com
quando vc digita instagram.com, 
ele nao vai direto para o servidor do instagram,
Ele vai diretamente para o servidor DNS, e pergunta:
Qual o IP publico atual do dominio do instagram.com?
Então ele devolve o numero IP do instagram 
para que o seu modem possa ter acesso ao servidor dele
então ele pede os arquivos do insta, e eles são
retornados por varias rotas, em varios pedacinhos
o que sao rotas?
uma rede finita, com caminhos finitos
nem sempre os arquivos virao pela rota mais rapida
assim como o WAZE, pode haver um congestionamento
então, ele pega uma rota diferente, para que seus
arquivos cheguem, por isso as vezes quando atualizamos
a pagina, ele faz uma nova rota para enviar, e assim
ter acesso aos arquivos que estavam preso em uma rota ruim
